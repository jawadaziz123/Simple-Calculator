# Simple Calculator

A basic calculator built with HTML, CSS, and JavaScript as a beginner project to practice core web development concepts.

## Features

- Perform basic arithmetic: addition, subtraction, multiplication, division, and modulo
- Clear button to reset the display
- Clean, responsive circular button design
- Real-time input display

## What I Learned

- DOM manipulation using `document.querySelectorAll()`
- Event handling with `addEventListener()` instead of inline `onclick` attributes
- Using `this` inside event listeners to identify the clicked element
- Basic error handling with `try...catch`
- Using `eval()` to evaluate string-based math expressions
- CSS Grid for layout, and Flexbox for centering

## Built With

- HTML5
- CSS3
- JavaScript (Vanilla, no frameworks/libraries)
